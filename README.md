## Why the name *Elementa Builder*?

**Elementa** is derived from the Latin *elementa* — the basic principles, first parts, or essential components of a thing.

That is exactly the idea behind this project.

**Elementa Builder** helps turn legal drafting into a structured process by organizing captions, sections, numbered paragraphs, and causes of action around the required elements of a claim. Instead of drafting from chaos, the goal is to build a complaint from the ground up: facts mapped to elements, elements assembled into counts, and counts formed into a coherent pleading.

The name was chosen because it reflects the core method of the app:

- identify the essential elements
- connect supporting facts to those elements
- build the complaint in an ordered way

**Elementa** is the short form and primary identity of the app.  
**Elementa Builder** is the fuller descriptive name for the tool as it exists today.
